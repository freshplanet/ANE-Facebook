init()method 
public function init(appID:String = null):void

Initialize the Facebook extension.

Parameters

appID:String (default = null) — A Facebook application ID (must be set for Android if there is missing FacebookId in application descriptor). <meta-data android:name="com.facebook.sdk.ApplicationId" android:value="fb{YOUR_FB_APP_ID}"/> NOTE: It is important to prefix YOUR_FB_APP_ID with "fb", because of bug in Android manifest file (http://stackoverflow.com/questions/16156856/android-facebook-applicationid-cannot-be-null). Facebook SDK code in this ANE was modified to recognize FB_APP_ID prefixed with "fb".

shareLinkDialog()method 
public function shareLinkDialog(contentUrl:String = null, contentTitle:String = null, contentDescription:String = null, imageUrl:String = null, useShareApi:Boolean = false, callback:Function = null):void

Open a native Facebook dialog for sharing a link This requires that the Facebook app is installed on the device, To make sure this succeeds, call canPresentShareDialog, otherwise you can fall back to a web view with the webDialog method

Parameters

contentUrl:String (default = null) — (Optional) Url to share.
 
contentTitle:String (default = null) — (Optional) Title of the publication.
 
contentDescription:String (default = null) — (Optional) Description of the link content.
 
imageUrl:String (default = null) — (Optional) Url of the attached picture.
 
useShareApi:Boolean (default = false) — If you have publish_actions permission you can directly post through ShareAPI.
 
callback:Function (default = null) — (Optional) A callback function of the following form: function myCallback(data:Object), where data is the parsed JSON object returned by Facebook.
